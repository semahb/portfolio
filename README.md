# Portfolio Project

This project is a showcase of my skills in backend development using Spring Boot 3.x, Java 17, Spring Data JPA, Liquibase, and OpenAPI documentation. The purpose of this project is to demonstrate how to build a robust web application with these technologies.

## Key Features
- **Spring Boot 3.x**: Utilizes the latest features and improvements of Spring Boot 3.x for building modern applications.
- **Java 17+**: Leverages the latest Java features and enhancements for better performance and security.
- **Spring Data JPA**: Implements data access using JPA to interact with databases efficiently.
- **Liquibase**: Manages database migrations using Liquibase, ensuring consistent database schema across environments.
- **OpenAPI 3.0 Annotations**: Provides comprehensive documentation for RESTful APIs through OpenAPI annotations.

## Getting Started
To get started with the project, follow these steps:
1. Clone the repository: `git clone https://github.com/yourusername/portfolio.git`
2. Navigate to the project directory: `cd portfolio`
3. Build the project using Maven: `mvn clean install`
4. Run the application: `java -jar target/portfolio-0.0.1-SNAPSHOT.jar`
