# Portfolio Project

This is a portfolio project created using Spring Boot 3.x, Java 17, Spring Data JPA, Liquibase, and OpenAPI documentation. The purpose of this project is to showcase my skills in building web applications with modern technologies.

Technologies used:
- Spring Boot 3.x
- Java 17+
- Spring Data JPA
- Liquibase
- OpenAPI 3.0 annotations
- Global exception handler
- DTOs for all request/response
- @Valid and Jakarta Bean Validation
- JUnit 5 + Mockito for tests
